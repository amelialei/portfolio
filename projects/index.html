Here are some of the projects I've worked on!

Cafe Recommender

A cafe recommendation system using Yelp datasets with 150k businesses and 6M reviews. It uses collaborative filtering and natural language processing for personalized recommendations.

  • Tools: Python, JupyterNotebook
  • Highlights: Data cleaning, NLP with TextBlob, K-Nearest Neighbors
  • GitHub: View Code

Weather App

A responsive web application displaying real-time weather data based on the user's IP address. It integrates multiple APIs and provides a clean, interactive interface.

  • Tools: Python, Django, HTML/CSS, Bootstrap
  • Highlights: API integration, real-time weather data, responsive design
  • GitHub: View Code

Recipe Exploratory Data Analysis

An exploratory data analysis of 83k recipes from food.com to understand recipe preferences and predict ratings. Features include fairness analysis and feature engineering for improved accuracy.

  • Tools: Python, JupyterNotebook, HTML
  • Highlights: Data cleaning, random forest classifier, fairness analysis
  • GitHub: View Code
